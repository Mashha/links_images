Links

Links are found in all pages. They can be an image or any other HTML element.

Anchor element ( a tag ) together with the href attribute creates a hyperlink. Without it it would be just plain text, and it wouldn't do anything.

In the example above, we added the attribute name, which is href and the value "../index.html"

The part that does not change one the website is the Go home text that you click.

We can add an anchor element on it's own or nest it in another HTML element.

We know absolute and relative links. Absolute links always include the domain name of the site with http://www; where relative links show the path to the file.


Images

Image element looks like this on it's own. It does not have a closing tag. Because it is not really a container where we would put text in.

We do use HTML attribute - src - as in source; which points to the URL of the image.

Just like with anchor element, we add name and value here too.
